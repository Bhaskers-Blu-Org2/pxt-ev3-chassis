# ev3-chassis

Controlling a bidirectional drive robot build using LEGO MINDSTORMS EV3.


## Usage

Call the ``drive`` block with the desired
speed, rotation speed and/or distance.

```blocks
chassis.largeBC.drive(10, 30, 0)
```
